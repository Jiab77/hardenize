# Hardenize
Just a script that helps to review and apply changes based on this excellent guide: https://madaidans-insecurities.github.io/guides/linux-hardening.html

## How to use

Simply run it as __root__ or with `sudo`:

```bash
$ sudo ./hardenize.sh
```
